Detect exclusionary or problematic language in your codebase

## Install

```bash
npm install -g ai-toxic-check
```

## Usage

```bash
npx ai-toxic-check ./src/
npx ai-toxic-check ./docs/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
